桁架有限元求解程序（总体刚度矩阵组装作业）
文件结构
main.py 主程序（同时运行算例 1、算例 2）
model.py 模型读取与前处理
element.py 单元刚度与 LM 矩阵计算
assembly.py 总体刚度矩阵组装
solver.py 方程求解与边界条件处理
postprocess.py 应力 / 轴力后处理
input_1d_bar.json 一维算例输入
input_2d_truss.json 二维算例输入
运行环境
Python 3.7+，依赖 numpy
bash
pip install numpy
运行方式
将所有文件放在同一文件夹，执行：
bash
python main.py
主要功能
自动生成对号矩阵 LM
直接组装总体刚度矩阵 K
缩减法处理位移边界
求解位移、约束反力、单元应力与轴力
输出对称性、奇异性、缩减刚度矩阵 Kff
输出内容
总体刚度矩阵 K 与对称性检查
节点位移、约束反力
奇异度检查与缩减刚度矩阵 Kff
单元长度、方向余弦、应力、轴力
二维算例自动输出 LM 矩阵
验证结果
算例 1：位移 [0, 0.1, 0.15]，反力 -10
算例 2：u3=38.284271，v3=-10.0，单元 1 应力 =-10，单元 2 应力≈14.1421
