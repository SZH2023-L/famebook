# 三维杆单元刚度矩阵计算程序 运行说明
## 运行环境
Anaconda Python 环境，使用 Jupyter Notebook 运行
依赖库：numpy（Anaconda自带，无需额外安装）

## 程序功能（与代码严格对应）
1. truss3d_element_stiffness：计算三维杆单元长度、方向余弦、6×6单元刚度矩阵，包含单元退化判断
2. truss3d_element_stress：由节点位移求解单元应变、应力、轴力
3. check_matrix_properties：检验刚度矩阵对称性、奇异性、半正定性、行列式与特征值
4. 内置2组验证算例、刚体位移检验、刚度矩阵物理意义验证

## 运行步骤
1. 打开Anaconda，启动Jupyter Notebook
2. 新建.ipynb文件，粘贴全部源代码
3. 直接运行全部代码，自动依次输出：
     算例1：沿x轴杆单元结果、刚度矩阵、一维杆退化验证、矩阵性质
     算例2：空间任意方向杆单元结果、刚度矩阵、矩阵奇异原因分析
     刚体位移检验结果
     刚度矩阵物理意义验证结果

## 注意事项
代码内置节点重合退化报错，可直接修改算例坐标、材料参数、位移条件进行计算。
